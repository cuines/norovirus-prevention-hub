# Norovirus Prevention Hub

A public, open-source repository for educational materials on norovirus prevention and control.

## Purpose

This repository serves as a reliable hub for clear, actionable resources aimed at preventing the spread of norovirus. It is designed for health educators, food service workers, childcare providers, and anyone interested in reducing norovirus outbreaks through evidence-based practices.

## Contents

We will host fact sheets, infographics, training materials, and other resources that explain:

- Norovirus symptoms and transmission
- Effective cleaning and disinfection protocols
- Hand hygiene best practices
- Food safety measures
- Outbreak response guidelines

## How to Contribute

We welcome contributions from health professionals, educators, and community members. Please review our contribution guidelines (to be added) and use GitHub Issues to suggest new resources or improvements.
